Twitter

During quarantine I launched a server that currently hosts four bots. This application is a part of an initiative I have to populate social media platforms with more "wholesome" content. Each bot is a painter, philosopher or writer and is programmed to take data from different APIs such as wikiMedia, goodReads, and poetryDb - perform some operations on the data - then post to its respective social media platform using its api. This project is growing and has bots added to it regularly.
